1. O Conceito de Protocolo

Computadores não têm intuição. Eles precisam de um algoritmo de comunicação.

  • É o idioma comum entre dispositivos diferentes.
  • Define como o dado é empacotado, endereçado e entregue.

2. Fundamentos do IPv4

O Endereço

Identificador único de 32 bits.

Permite 4.294.967.296 endereços.

Os Octetos

Dividido em 4 grupos de 8 bits.

Cada grupo varia de 0 a 255.

3. Localhost (127.0.0.1)

O endereço de Loopback. É o seu computador conversando com ele mesmo sem sair para a placa de rede física.

Para que serve?

  • Testar servidores locais (Dev).
  • Garantir que a "pilha TCP/IP" está funcionando.

5. Máscara de Rede & CIDR

A máscara separa o que é Identidade da Rede e o que é Identidade do Host.

Exemplo: /24 (255.255.255.0)

Significa que os primeiros 24 bits estão "trancados" para a rede e os últimos 8 bits são para os seus aparelhos.

6. IP Público vs Privado

Público

Visível em toda a Web. Único. Fornecido pelo Provedor.

Privado

Invisível para a Web. Reutilizável em outras redes (NAT).

7. Cálculos de Sub-Rede

Fórmula de Hosts: $2^h - 2$

Onde h é o número de bits de host (os bits 0 na máscara).

Subtraímos 2 porque o 1º IP é a Rede e o último é o Broadcast.
